PDDA: A Project for Personal Digital Device Assistants

An open source project of software libraries towards a general-purpose GUI control model

RecGUI

Environment for recording GUI operations. It provides a uniform way to record screenshots and actions in the GUI (pointer and key events) for the creation of training data.

The environment consists of a Docker container where running the Ubuntu OS, a target application (e.g. browser), and recording utilities.

Technical note: please refer to this link to see the detail. Currently, the japanese version is only available.

Public records: we make some records created with the environment publicly available.
